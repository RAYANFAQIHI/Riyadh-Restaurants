## Overview
An interactive **Power BI dashboard** built to analyze restaurant data in Riyadh.  
This project explores key metrics such as total places, rated vs. unrated restaurants, price distribution, top categories, and average ratings.


## Objectives
- Explore Riyadh restaurant data
- Analyze rating coverage
- Study price-level distribution
- Identify top restaurant categories
- Build an interactive dashboard for data-driven insights

## Tools Used
- Power BI
- Python
- Pandas
- Jupyter Notebook

## Files
- `Riyadh Restaurants.pbix`
- `riyadh_restaurants_all.csv`
- `Riyadh_Resturants.ipynb`

## Key Features
- KPI cards for restaurant overview
- Price distribution analysis
- Top categories visualization
- Average rating by price level
- Interactive filters and restaurant table

## Outcome
This dashboard helps present Riyadh restaurant data in a clear and interactive way, making it easier to identify trends and insights.
